Getting Started

The PX Submission Tool is a portable, cross-platform application that makes it easy to submit proteomics datasets to the ProteomeXchange repository. This guide will help you get started.

  • Download and Installation
  • Launching the Tool
  • Troubleshooting

Download and Installation

The PX Submission Tool is distributed as a portable ZIP file that works on Windows, macOS, and Linux.

Download: Get the latest version from the GitHub Releases page.

Extract: Simply extract the ZIP file to any location on your computer. No installation is required - the tool is completely portable.

File Structure: After extraction, you'll see:
  • px-submission-tool-2.11.1.jar - Main application
  • start.bat - Windows launcher
  • start.sh - macOS/Linux launcher
  • README.md - Detailed documentation

Launching the Tool

The tool includes smart Java management that automatically handles Java requirements:

Windows: Double-click start.bat or run it from Command Prompt.

macOS: Open Terminal, navigate to the tool directory, and run ./start.sh.

Linux: Open Terminal, navigate to the tool directory, and run ./start.sh.

Smart JRE Management:
  • Checks if Java 21+ is already installed on your system
  • If not found, automatically downloads Java 21 JRE
  • Downloads are cached - won't download again on subsequent runs
  • Works offline after initial setup
macOS Security: If you get security warnings on macOS:
  • Right-click the launcher and select "Open"
  • Or run: xattr -cr . in Terminal to remove quarantine attributes

Troubleshooting

Java Issues: If the tool can't find or download Java:

  • Install Java 21 manually from Adoptium
  • Ensure your internet connection is working
  • Check firewall settings if behind a corporate network

Permission Issues: If you get permission errors:

  • Windows: Run Command Prompt as Administrator
  • macOS/Linux: Run chmod +x start.sh to make the script executable

Network Issues: If the tool can't connect:

  • Check your internet connection
  • Verify firewall settings
  • Try using a different network (mobile hotspot)
