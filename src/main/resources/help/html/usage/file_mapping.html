How to add related files?

This panel contains two main tables, the top table lists all the result/search files in your submission. For each file, it is mandatory to provide related files.

You can add related files by clicking the Add Relation button, as highlighted below:

A dialog will popup to allow you to choose the file by clicking on the checkbox or clicking on a row in the table.
