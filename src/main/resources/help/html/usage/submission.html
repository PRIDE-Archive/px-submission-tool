The final submission panel will upload the files to the ProteomeXchange and show the progress of your upload. Once the submission is done, a submission reference will be shown, and a confirmation email will be sent to you.

  • Submission In Progress
  • Submission Finished

The submission reference is not a ProteomeXchange accession, it is a temporary ID to help support any questions before the dataset is officially committed to the ProteomeXchange.

Transfer Methods: The tool supports multiple transfer methods:
  • Aspera - Fast transfer (recommended)
  • FTP - Standard file transfer
  • Globus - Alternative for restricted networks

Error Handling

If you encounter transfer errors, the tool provides helpful error messages and suggestions:

Network Issues: If Aspera or FTP transfers fail due to network restrictions or instability, the tool will suggest using Globus as an alternative transfer method.

Error Messages: Error messages are displayed in a scrollable text area with proper formatting to ensure you can read the complete error details.

Retry Logic: The tool includes automatic retry logic for transient network issues.

Common Solutions:
  • Check your internet connection
  • Try a different transfer method (Aspera → FTP → Globus)
  • Contact your network administrator if behind a firewall
  • Use Globus for large datasets or restricted networks
