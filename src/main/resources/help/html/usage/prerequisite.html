Prerequisite

Previous | Next | See Also
Before starting the submission, you need to check that you have the required information. Please note that there are different requirements for different submission types.
System Requirements:
  • Java 21 or later - The tool will automatically download Java 21 JRE if not available
  • Internet connection - Required for file uploads and Java download
  • Operating System: Windows 10+, macOS 10.14+, or Linux

Complete submission

The complete submission requires three sets of information:

  • Result Files: they are either mzIdentML or mzTab with spectrum files such as: MGF, mzML.
  • Raw Files: MS instrument outputs, such as: BRUKER .baf files. If your raw outputs are directories instead of files, please compress them into individual files before upload.
  • PRIDE Login: PRIDE user name and password. All submissions to the ProteomeXchange are private by default, the user name and password will help us to secure your submissions. You can register a new user account at PRIDE
  • Experiment Details: Details about the setup and context of your experiment. e.g., sample details, protocols, species, tissues
  • Lab Head: The affiliation of the lab head. e.g., name, email, and address

    • Although not required, we would also recommend you to submit the following data files:

  • Search result files: the original output from your search engine or your analysis pipeline, such as: Mascot .dat files. They should contain the peptide/protein identifications.
