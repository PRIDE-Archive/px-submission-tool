How to remove files?

You can remove files in two ways:

Individual file removal: When you have files in the table, a "Remove" column will appear with trash icon buttons. Click the trash icon next to any file to remove it individually.

Remove all files: Click the "Remove Files" button to remove all files at once. Note that checksum.txt files are protected and cannot be removed.

Important: checksum.txt files are automatically generated and protected from removal to maintain submission integrity.

Checksum files

The tool automatically generates checksum.txt files to ensure data integrity during upload. These files:

  • Are created automatically when you add files
  • Contain checksums for all your data files
  • Are protected from accidental removal
  • Are required for successful submission

You don't need to manually manage checksum files - the tool handles them automatically.
