Dataset Details

The dataset details section requires six key pieces of information about your project:

  • Poject Title: A title about your project, we suggest you to use the title to describe the your experiment.
  • Keywrods: Keywords that describes the nature of your experiment, for example: you can put the species, tissue used in your experiment.
  • Project Description: This gives you the opportunity to add a short paragraph to describe your experiment and datasets. Please provide an overall description of your study, think something similar in scrope to the manuscript abstract.
  • Sample processing protocol: A short description of the sample processing protocol being followed. Please provide a short description on the sample preparation steps, separation, enrichment strategies and mass spectrometry protocols included.
  • Data processing protocol: A short description of the data processing protocol being followed. Please provide a couple of sentenses on the bioinformatics pipeline used, main search parameters, quantitative analysis, software tools and versions included. Think something similar in scope to the Data Analysis section of your manuscript.
  • Experiment method: Select from a predefined list of experiment methods.
