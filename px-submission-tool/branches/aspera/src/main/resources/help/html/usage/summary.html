Submission summary

The submission summary panel on the top of this screen is to give you a quick overview on the files you are about to submit to ProteomeXchange. They are shown in file counts, one total file count and one file count for each permitted file type.

File to upload

A table which lists all the files to be submitted, it also shows the relationship between different files in a tree structure. Click on the expandable nodes to see the source files related to the result file.
