Additional dataset details

Previous | Next

The Additional Dataset Details step asks you to give some additional details about your project, all of them are optional but recommended if available.

The additional dataset details section requires 4 optional pieces of information about your project:

  • Parent project: Select parent project from the table if your dataset is part of a large project.
  • PubMed ID(s): Provide the PubMed ID(s) if the dataset is associated with an existing publication
  • Reanalysis ProteomeXchange accession(s): Only applicable if your results are based on the reprocessing of one or several previously submitted PX dataset(s)
  • Links to other 'Omics' datasets: Only applicable if proteomics results can be linked to other biological data submitted to other resources (e.g. ArrayExpress, GEO)
