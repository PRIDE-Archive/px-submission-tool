The final submission panel will upload the files to the ProteomeXchange and show the progress of your upload. Once the submission id done, a submission reference will be shown, and an confirmation email will be sent to you.

  • Submission In Progress

The submission reference is not ProteomeXchange accession, it is a temporary ID to help to support any questions before the dataset is officially committed to the ProteomeXchange.
